# Судоку в мягком сказочном стиле

Простая браузерная игра Судоку на чистом HTML/CSS/JavaScript.

## Что реализовано

- Поле 9x9 и блоки 3x3
- Фиксированные и редактируемые клетки
- Уровни сложности: `Легкая`, `Средняя`, `Сложная`
- Ввод с клавиатуры: `1-9`, очистка `Backspace/Delete/0`
- Проверка конфликтов по правилам судоку:
  - строка
  - столбец
  - блок 3x3
- Таймер текущей партии
- Счетчик ошибок (увеличивается при конфликтном вводе)
- Подсветка ошибок
- Проверка завершения с сообщением о победе
- Кнопки: `Новая игра`, `Проверить`, `Сбросить`

## Локальный запуск

### Вариант 1 (самый простой)

Откройте файл `index.html` в браузере.

### Вариант 2 (через локальный сервер)

Из папки проекта:

```bash
python3 -m http.server 8000
```

Затем откройте [http://localhost:8000](http://localhost:8000).

## Публикация как сайта

Это полностью статический проект, для публикации достаточно файлов:

- `index.html`
- `styles.css`
- `app.js`

### GitHub Pages

1. Создайте репозиторий и загрузите файлы.
2. Включите GitHub Pages в настройках репозитория (ветка `main`, корень `/`).
3. Получите публичную ссылку на игру.

### Netlify / Vercel

1. Импортируйте репозиторий или загрузите папку вручную.
2. Тип проекта: Static Site.
3. Build command не нужен, publish directory: корень проекта.
