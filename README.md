# Судоку: Лунтик 2049

Браузерная игра Судоку на чистом `HTML/CSS/JavaScript` в стилистике `Лунтик 2049`.

## Функциональность

- Поле `9x9` и блоки `3x3` по стандартным правилам судоку
- Фиксированные и редактируемые клетки
- Уровни сложности: `Легкая`, `Средняя`, `Сложная`
- Ввод с клавиатуры: `1-9`, очистка: `Backspace/Delete/0`
- На мобильных устройствах при тапе по пустой клетке появляется блок выбора цифр `1-9` и кнопка очистки
- Подсветка:
  - корректный ввод — зеленый
  - конфликтный ввод — красный
- Таймер партии и счетчик ошибок
- Победа с итогом по времени и ошибкам
- Показ спец-изображения после каждых 2 ошибок подряд (с авто-скрытием)
- Звуковые эффекты (кнопки, правильный/ошибочный ввод, победа)
- Синтезируемая фоновая sci-fi музыка:
  - `Музыка: вкл/выкл`
  - отдельный ползунок громкости
- Раздел с правилами игры
- Адаптация под мобильные устройства

## Локальный запуск

### Быстрый запуск

Откройте `index.html` в браузере.

### Через локальный сервер

```bash
python3 -m http.server 8000
```

Откройте [http://localhost:8000](http://localhost:8000)

## Публикация

Проект полностью статический. Достаточно загрузить:

- `index.html`
- `styles.css`
- `app.js`
- папку `assets`

### GitHub Pages

1. Создайте `public`-репозиторий.
2. Загрузите все файлы проекта.
3. В `Settings -> Pages` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
4. Сохраните и дождитесь публикации (обычно 1-3 минуты).

### Netlify / Vercel

1. Импортируйте репозиторий.
2. Тип: `Static site`.
3. Build command не нужен, publish directory — корень проекта.

## Технологии

- `HTML5`
- `CSS3` (адаптивная верстка, анимации, неоновая тема)
- `Vanilla JavaScript`
- `Web Audio API` (SFX + фоновая синтез-музыка)
